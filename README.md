# minit-check
minit-check is an simple c program aimed to determine which init system is being used, then it prints the name of the init. 

## why
 - ricing
 - fast
 - reliable
 - not rust

## init systems suported (adding further support soon)

 - systemd
 - sysvinit
 - upstart
